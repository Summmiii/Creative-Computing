THE END  

“Information is only useful when it can be understood.” We’ll leave the last word on her design philosophy to Cooper–who was never shy about expressing herself. “Electronic is malleable. Print is rigid. I guess I’m never sure that print is truly linear….Designers know a lot about how to control perception, how to present information in some way that helps you find what you need, or what it is they think you need. Information is only useful when it can be understood.”
